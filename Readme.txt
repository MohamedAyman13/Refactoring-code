# My Advanced Blog

## Overview
My Advanced Blog is a responsive HTML website that showcases a blog layout with a dynamic navigation bar, multiple articles, a sidebar, a contact form, and a footer. The website is designed using Bootstrap, a popular CSS framework, and includes custom CSS and JavaScript for additional functionality.

The website features a clean and modern design, with a focus on readability and user experience. The articles on the home page are displayed in a card format, with a title, excerpt, and a "Read More" button. The sidebar provides additional information and links to other pages. The contact form allows users to submit their email address, password, and a checkbox for additional information.

## Programming Language Used
The website is primarily built using HTML, CSS, and JavaScript. Bootstrap is used for the responsive design and layout. Custom CSS and JavaScript are included for additional functionality, such as the back-to-top button.
